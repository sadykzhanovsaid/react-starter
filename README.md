# 🚀 React Starter

Minimal and modern React starter powered by Vite.
Preconfigured for fast development with a clean and scalable structure.

## ✨ Features

* ⚡ Vite for lightning-fast builds
* ⚛️ React setup
* 📁 Clean project structure
* 🔗 Path alias (`@/`)
* 🖼 SVG support as React components (SVGR)
* 🧹 Ready for scalable development

## 📦 Installation

```bash
git clone https://github.com/your-username/react-starter.git
cd react-starter
npm install
npm run dev
```

## 🛠 Tech Stack

* React
* Vite

## 📌 Purpose

This starter is designed for:

* landing pages
* small to medium React projects
* portfolio development
